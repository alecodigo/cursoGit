Curso de Git y GitHub

Como dijo un sabio empecemos por el principio.

Que es git?

Git es un sistema de control de versiones muy utilizado hoy en dia en la industria del desarrollo del software y creado por Linus Torvalds. Este sistema nos permite crear instantaneas de los distintos estados en los que se pueda encontrar nuestro codigo y es a esto lo que llamamos versiones.

Que es un sistema de control de versiones?

Un control de versiones es un sistema que registra los cambios realizados a un archivo o conjunto de archivos en el tiempo de modo que puedas recuperar versiones especificas de tu codigo.

Como funciona git?

Git es un programa como cualquier otro que puede ser instalado en tu pc de escritorio (tambien se pude instalar en un servidor) la instalacion puede variar dependiendo del sistema operativo que estes utilizando para efectos de este curso vamos a trabajar sobre Ubuntu.

Como instalar Git

En Ubuntu puedes instalar git desde la terminal ingresando la siguiente instruccion: apt-get install git y el programa se descargará del repositorio y se instalará en tu pc.

Y.. luego que?

Empezamos a Configurar

  1. Correo
  2. Nombre de usuario
  3. Editor de texto

Debemos configurar nuestra instalacion agregando nuestro correo electrónico y nuestro nombre para que git pueda identificar el autor en los commits (Mas adelante explicaremos que son los commits).

Ingresa en la terminal el siguiente comando: git config user.email "Tu correo"

Ingresa en la terminal el siguiente comando: git config user.name "Tu nombre"

Ingresa en la terminal el siguiente comando: git config core.editor "Tu nombre"
